一、文献简介

读了《The Linux Command Line A Complete Introduction》的前两章，我理解和掌握了以下知识：
1.命令历史记录：如果按下键盘的向上箭头，就可以看到上一条执行的命令重复出现。按下向下箭头，前面的命令就会消失。左右箭头可以将光标定位在命令行的任意位置。
2.快速复制粘贴：如果通过按住鼠标左键并将鼠标拖动在其上（或双击一个单词）来突出显示某些文本，它将复制到x维护的缓冲区中。按下鼠标中键将导致文本粘贴在光标位置。不要使用ctrl-C和ctrl-V在终端窗口内执行复制和粘贴，没有用。
3.命令date：能够显示当前的时间和日期。
4.命令cal：默认情况下，它会显示当前月份的日历。
5.命令df：查看磁盘驱动器上的当前可用空间量。
6.命令free：显示空闲内存的数量。
7.命令pwd：显示当前的工作目录。
8.命令ls：列出当前工作目录中的文件和目录。
9.命令cd：更改工作目录。
10.虽然Linux支持可能包含嵌入式空格和标点字符的长文件名，但需将创建的文件名称中的标点字符限制为句点、破折号和下划线。最重要的是，不要在文件名中嵌入空格。如果要在文件名中表示单词之间的空格，请使用下划线字符。

三、课堂总结

从课堂上我学到了以下知识：
1.命令mkrid：通过 mkdir 命令可以实现在指定位置创建以 DirName(指定的文件名)命名的文件夹或目录。要创建文件夹或目录的用户必须对所创建的文件夹的父文件夹具有写权限。并且，所创建的文件夹(目录)不能与其父目录(即父文件夹)中的文件名重名，即同一个目录下不能有同名的(区分大小写)
2.rm ( remove )：删除文件
3.-rm -rf 文件夹：（循环递进删除文件夹,不需要二次确认）
4.mv ( move )：移动文件或重命名
5.cp ( copy )：复制文件，要注意后面有空格加点
6.pwd ( print working directory )：查看当前所在路径--绝对路径
7.cd ( change directory )：切换目标
8.cd .. ：返回到上一个目录
9.ls ( list )：查看当前目录下的内容
10.ls -R命令：能够列出当前工作目录下所有文件及目录并以文件的大小进行排序
11.cat [文件名]：一次显示整个文件
12.ls -lh：人性化显示是目录下每个文件的详细信息 , h相当于humanize

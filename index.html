7.cd ( change directory ) 切换目标
8.cd .. 返回到上一个目录
9.ls ( list ) 查看当前目录下的内容

文献所学知识：

读了《The Linux Command Line A Complete Introduction》的第三章，我理解和掌握了以下知识：
1.ls 查看文件内容，确定各种重要的文件和目录属性
2.file 确定文件类型
3.less 查看文件内容，退出查看
4.ls -l：更改输出格式为长格式来显示更多的细节
5.ls -lt:ls命令有两个选项，一个是生成长格式输出的l选项，另一个是根据文件的修改时间对结果进行排序的t选项
6.ls -lt --reverse:添加长选项，反向来反转排序的顺序

ls命令有大量可能的选项，其中最常见的选项如下：
-a --all:列出所有的文件，即使是那些名称以句号开头的文件，它们通常不被列出（即隐藏）
-A --almost-all：就像-a选项一样，除了它没有列出当前目录和..父目录。
-d --directory：可以查看有关目录的详细信息，而不是其内容
-F --classify：此选项将在每个列出的名称的末尾添加一个指示符字符
-h --human-readable：在长格式列表中，以人可读格式显示文件大小，而不是以字节格式显示文件大小。
-l：以长格式显示结果。
-r --reverse：以相反的顺序显示结果。通常，ls以字母顺序升序显示其结果。
-S：按文件大小对结果进行排序。
-t：按修改时间排序

ls长列表字段：
-rw-r--r--：访问该文件的权限。第一个字符表示文件的类型。在不同的类型中，前导破折号表示常规文件，而d表示目录。后面的三个字符是文件所有者的访问权限，后面的三个字符是针对文件组的成员的，最后三个字符是针对其他所有人的。

第六周作业

课后任务：

完善个人播放器：
1.输入秒数，实现当前视频的进度跳转；
2.输入视频序号，实现视频跳转；

涉及知识及实现过程：

涉及知识如下：
1.HTML：了解HTML的基本语法和标签，以及如何创建表单元素（如输入框和按钮）和如何使用JavaScript与HTML元素进行交互。
2.JavaScript：掌握JavaScript的基本语法和DOM操作，以便能够根据用户输入来更新视频播放器的状态和进度。
3.HTML5 Video API：了解HTML5的Video API，以便能够控制视频的播放、暂停、进度和其他相关功能。
具体实现过程可以如下：
1.创建一个HTML表单，包含输入框和按钮元素，用于接收用户输入的秒数和视频序号。
2.在JavaScript中，为输入框添加事件监听器，以便在用户输入时捕获并处理输入值。
3.根据用户输入的值，使用HTML5 Video API控制视频播放器的状态和进度。如果用户输入了秒数，则将播放器的currentTime设置为相应的秒数，实现进度跳转。如果用户输入了视频序号，则加载相应的视频并播放。
